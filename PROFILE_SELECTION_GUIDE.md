# Profile Selection Guide

Profiles are curated skill lists. They are not native Antigravity configuration files. Use them with `scripts/install_profile.py`.

## Suggested order

1. `minimal_core`
2. one specialist profile:
 - `builder_engineering`
 - `research_operator`
 - `docs_support`
 - `data_analytics`
 - `design_product`
 - `security_quality`
3. add a second specialist profile only after the first proves useful

## Use `full_library` only when
- you understand the catalog
- you want a broad exploratory install
- you accept a noisier skill surface
