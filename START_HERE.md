# Start Here

This library is a **RexBot / Rex Hub community skill pack for Antigravity**.

## First run

1. Read `README.md`
2. Pick a profile from `PROFILE_SELECTION_GUIDE.md`
3. Install it into `.agent/skills` for one repo or `~/.gemini/antigravity/skills` for user-wide use
4. Add always-on project guidance in `AGENT_NOTES.md` or a local project note
5. Start a fresh session and invoke one skill explicitly first

## Fastest safe path

- Install `minimal_core`
- Add one specialist profile
- Keep high-impact changes routed through safety, quality, and security skills
- Expand only when the current setup proves useful

## Explicit invocation

Use `@skill-name or /skill-name`.

## Why this pack is structured this way

This library is broad, but it is intended to stay usable by loading only the skills a user or project actually needs. The profiles are there to keep your skill surface lean while still giving you deep playbooks when you need them.
